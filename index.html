MISSION & VISION

"お祭り"を通じた
地方創生の実現

担い手の減少、関係人口の縮小、運営の高齢化。"お祭り"が抱える課題は、地域の課題そのもの。
WASSYOI は、運営の足腰をデジタルで支えることで、地域に根ざした"お祭り"の持続可能性を底上げします。

VALUE 01

現場にねざして

"お祭り"で必要な仕組みは普通のデジタルツールとは少し違います。現場の声から生まれたツールは、現場の皆さんの声でさらに進化していきます。

VALUE 02

私たちは裏方

主役はあくまで"お祭り"の現場とその参加者の皆さんです。私たちは面倒な部分を引き受け、"お祭り"そのものに集中していただくための裏方です。

VALUE 03

横につながる"お祭り"DX

複数の"お祭り"の運営を経て得たノウハウ、情報がツールに込められています。新しく"お祭り"を始める方にはもちろん、ベテランの皆様にも実は役立つツールです。

CONCEPT

"お祭り"は
地域の熱量の象徴

WASSYOI は、地域の"お祭り"の運営をデジタルで支える学生ベンチャーです。
メンバー自身も仙台で"お祭り"を運営してきた経験を活かし、現場のリアルな課題を解決します。

ふるさとの記憶、地域のつながり、文化の継承——
伝統的な祭礼から、地域に根ざした恒例イベントまで、
すべての"お祭り"は、その土地の熱量の象徴です。
