02 — PROJECTS

つくっている、お祭り。

◉ 2 件 IN PRODUCTION
NO.001 — JAZZ / SENDAI

定禅寺ストリートジャズフェスティバル
ボランティア管理

宮城県仙台市・2026.09.12（土）— 09.13（日） / 第35回
2026年 開催に向け運用中 v1 — α release

約 1,000 人のボランティアを、街中 50 拠点に効率配置するための運営システム。 ボランティア募集 → 配置 → 当日運用 → 事後フォローまでを一気通貫で支援し、 「当日の混乱（欠員・再配置・連絡地獄）」を最優先で解消。

時間帯×エリアの希望者分布をリアルタイムで把握できる管理ダッシュボードと、 サンキー図で「継続／新規／離脱」の人の流れを見える化する設計が特徴。

つかっている技術

Next.js 16 TypeScript Tailwind CSS 4 Radix / shadcn-ui Supabase Resend reCAPTCHA v3

NO.002 — YAPPO / ISHINOMAKI

ほやっほー祭 2026
ボランティア募集サイト

石巻市かわまち交流広場・2026.07.04（土）— 07.05（日）
2026年 開催に向け運用中 deployed on Vercel

石巻のローカル祭「ほやっほー祭」のボランティア募集 → 配置 → 当日運用までを担う Web アプリ。 応募フォーム、運営ダッシュボード、ボランティア向けマイページ、当日の QR チェックインを備える。

Supabase は RLS deny-all + Server Action（service_role）経由のみ書込可、という安全側に倒した設計。 応募 ID は HOYA-2026-NNNN 形式でシーケンス採番、 応募〜マイページ〜当日運用まで同じ DB を一気通貫で使う。

Next.js 16 React 19 TypeScript Server Actions Supabase（RLS） Cloudflare Turnstile Vercel

03 — ABOUT

WASSYOIって何？

◉ 仙台 / 石巻

WASSYOI は、地域の祭・イベントの運営をデジタルで支える小さな工房です。 主役はあくまで現場で動く実行委員とボランティア、 私たちは応募から当日運用までの「面倒な部分」を引き受けて、お祭そのものに集中してもらうのが仕事です。

得意分野は、応募フォーム・配置最適化・当日のシフト運用・QRチェックインといった、 お祭り単位で繰り返し必要になる仕組み。 Next.js と Supabase をベースに、お祭ごとに UI を仕立てて納品しています。

現在は東北を中心に、定禅寺ジャズフェス（仙台）と ほやっほー祭（石巻）の運営を支援中。

WHAT WE BUILD
  • 大規模ボランティア募集〜配置の管理システム
  • 運営者向けの可視化ダッシュボード（時間帯×拠点）
  • 応募者マイページとシフト確認・QRチェックイン
  • 主催者向けの当日運用ツール（自動編成・通知・CSV）
